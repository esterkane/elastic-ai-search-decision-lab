# Project Summary

`elastic-ai-search-decision-lab` is a compact TypeScript, Node.js, and Elasticsearch portfolio project for documentation search quality.

It indexes six AI search decision-guide Markdown drafts, enriches them with deterministic metadata, compares baseline search against metadata-aware search and routed search, then evaluates findability with practitioner-style judgment queries.

The project demonstrates Markdown parsing, frontmatter handling, Zod validation, Elasticsearch mappings, bulk indexing, ranked search strategies, deterministic intent routing, Precision@1, MRR@3, nDCG@3, and JSON/Markdown report generation.

It is independent portfolio work, not official Elastic documentation, and intentionally avoids a frontend, chatbot, RAG workflow, and external LLM APIs.
